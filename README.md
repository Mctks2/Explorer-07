# Calculadora IMC

Página web interativa que permite ao usuário calcular seu IMC. Caso não insira números nos campos, use números com vírgula ou deixe em branco, um alerta de erro será exibido.

## 🛠️ Tecnologias utilizadas
- HTML
- CSS
- JavaScript
